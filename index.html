Project Overview

What is this project about?

An end-to-end machine learning system that automatically recognizes handwritten digits using state-of-the-art deep learning techniques.

Image Preprocessing

Normalizes pixel values, removes noise, and resizes input images to a standard 28×28 grayscale format for consistent model input.

CNN Architecture

Convolutional Neural Networks automatically learn edges, curves, and textures from raw pixel data, delivering near-human recognition accuracy.

Model Evaluation

Comprehensive benchmarking using Accuracy, Precision, Recall, and F1-Score across SVM, MLP, and CNN architectures.

Real-Time GUI

An interactive graphical user interface enables users to draw digits and receive instant predictions from the trained model.
